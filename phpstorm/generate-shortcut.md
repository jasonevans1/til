# Shortcuts to generate code.

Generate a constructor.
* Highlight class variables.
* Press command N .
* Select Constructor...
  * Select the variables.
* Press enter.

Generate phpdoc
* Press command N .
* Select phpdoc.
  * Select the variables.
* Press enter.
